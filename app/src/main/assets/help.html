How to use?

  • This application is usefull for current city weather details.
  • First time user can see few predefind city in home screen,if user wants to add new city in home screen list go to map screen and add new city from this screen.
  • Click on city name new screen will shown details like temprature,humidity,Rain Timing,Wind speed etc.
  • User can delete bookmark city from home screen using "Remove" button click.
  • User can clear all bookmark city list from Profile screen.
